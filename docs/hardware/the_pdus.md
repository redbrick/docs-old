#### Web Interface

1. Issue the command
   `sudo ssh -L 443:192.168.1.70:443 yourname@b4.redbrick.dcu.ie`
2. Fire up a web browser
3. Enter the URL [localhost](https://localhost)
4. Login with the details in the pwsafe
